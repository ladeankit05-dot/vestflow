# VestFlow — Token Vesting Contract Generator (MVP skeleton)

This repo is a pnpm monorepo with:
- `apps/web` — Next.js frontend (placeholder)
- `packages/contracts` — Solidity contracts with Foundry config
- `packages/shared` — shared TypeScript types and ABIs

This skeleton is designed for cloud-only development using GitHub Codespaces.

See `.github/` for CI and PR templates.
